1) Pengantar

Aplikasi Cooliner ‚Äì Kasir Kuliner Offline ("Aplikasi", "Kami") berkomitmen melindungi data yang Anda masukkan. Kebijakan ini berlaku untuk semua pengguna aplikasi.

2) Pengumpulan Informasi

Aplikasi ini tidak mengirimkan data Anda ke server kami dan bekerja secara offline-first. Semua informasi operasional tetap tersimpan lokal di perangkat Anda.

Contoh data yang mungkin Anda simpan di perangkat:

  • Data penjualan dan transaksi
  • Data produk, harga, stok
  • Data pelanggan (opsional)
  • Pengaturan & preferensi aplikasi

Catatan: Jika Anda mengaktifkan fitur backup/ekspor, file akan tersimpan ke penyimpanan perangkat atau lokasi yang Anda pilih (mis. Google Drive melalui aplikasi pihak ketiga yang Anda gunakan).

3) Penggunaan Informasi

  • Menyediakan fungsi kasir dan manajemen produk
  • Menyimpan riwayat transaksi & laporan
  • Membantu analisis sederhana untuk operasional bisnis

Kami tidak menjual, menyewakan, atau membagikan data Anda kepada pihak ketiga untuk tujuan pemasaran.

4) Penyimpanan & Keamanan Data

  • Data disimpan lokal di perangkat Anda.
  • Anda bertanggung jawab melakukan backup berkala.
  • Kami menyarankan melindungi perangkat dengan PIN/sandi/biometrik.
  • Jika perangkat hilang/rusak, data mungkin tidak dapat dipulihkan tanpa cadangan.

5) Hak Anda

  • Akses: melihat data di dalam aplikasi.
  • Ekspor: mengekspor data untuk cadangan atau migrasi.
  • Hapus: menghapus data atau mereset aplikasi kapan saja.

6) Izin Aplikasi

  • Penyimpanan: menyimpan/memuat file cadangan.
  • Kamera (opsional): pemindai barcode atau foto produk.

Izin hanya dipakai untuk fungsi tersebut dan tidak digunakan untuk mengumpulkan data pribadi tanpa persetujuan Anda.

7) Layanan Pihak Ketiga

Aplikasi inti tidak mengirim data ke server. Jika Anda memilih menggunakan layanan pihak ketiga (misalnya, mengunggah file backup ke layanan cloud melalui aplikasi terpisah), penggunaan Anda pada layanan tersebut tunduk pada kebijakan privasi masing-masing penyedia.

8) Perubahan Kebijakan

Kami dapat memperbarui kebijakan ini dari waktu ke waktu. Versi terbaru akan tersedia di halaman ini dan/atau di dalam aplikasi saat pembaruan dirilis.
