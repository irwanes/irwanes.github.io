Tutorial Reset Data Aplikasi OKasir - Owner & Kasir

Jika kamu ingin menghapus data pada aplikasi OKasir - Owner, silakan ikuti tutorial berikut. Karena data OKasir - Owner dan OKasir - Staff terintegrasi, maka untuk menghapus data pada OKasir - Staff dilakukan dengan menghapus data pada OKasir - Owner

Data yang dihapus adalah semua data yang sudah anda input pada aplikasi, yaitu data produk, data penjualan, data kasir dan lain-lain

Perlu anda perhatikan bahwa data yang sudah anda hapus tidak bisa dikembalikan

1. Buka Aplikasi lalu masuk ke menu Pengaturan

2. Setelah masuk ke menu Pengaturan, pilih menu Reset Data

3. Pilih data apa saja yang akan dihapus, lalu klik Reset Data

4. Untuk menghapus akun, pilih menu Hapus Akun

5. Pilih Hapus Akun

6. Selesai
